## ğŸš€ Quick Start

1. **Clone and setup**
   ```bash
   git clone <your-repo-url>
   cd hidaya
   chmod +x setup.sh
   ./setup.sh
   ```

2. **Add OpenAI API key**
   ```bash
   # Edit backend/.env
   OPENAI_API_KEY=your_api_key_here
   ```

3. **Start the app**
   ```bash
   # Terminal 1: Backend
   cd backend && source venv/bin/activate
   uvicorn main:app --reload --host 0.0.0.0 --port 8000
   
   # Terminal 2: Flutter App
   flutter run
   ```
